# LLM-From-Scratch

I created a simple bigram-LLM from scratch and trained it on Shakespeare text. It learns to predict the next character based on the previous one, giving a basic understanding of how language models work.

This project is inspired by  
[Attention Is All You Need] (https://arxiv.org/abs/1706.03762).

The goal of this project is to build strong fundamentals before moving to more advanced architectures like Transformers.
